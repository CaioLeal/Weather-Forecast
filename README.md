### â­ How to use this project locally?

* To run this project on your machine, I recommend **cloning** this repository. It is not necessary to install Node.js, as the project uses native technologies.

```bash
git clone https://github.com/CaioLeal/Weather-Forecast.git/
```
Right after that, open the project folder in your IDE (VSCode) or navigate to the folder: ```index.html```

*Important*: Make sure that the API key in the **script.js** file is properly configured for the search to work.
